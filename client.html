TUS golang server with tus-js-client example

This demo shows the basic functionality of the tus protocol. You can select a file using the controls below and start/pause the upload as you wish.

For a prettier and more official demo please go to the tus.io website.

This is an all-in-one example to aid developers with both a TUS client & TUS server working in one executable.

This example is based upon modified versions of TUS golang server example and tus-js-client

Warning: This code is not production worthy, it is just a "get'er work'n" example!
